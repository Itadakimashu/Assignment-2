Discuss the scope of var, let and const

var

  • Variables declared with var keyword are only accessible in the function scope where they are declared in. var variables are hoisted to the top of the function scope where they are declared in.

let

  • The let Variables are accessible in the block they are declared which can be anything defined with curly braces. let Variables cannot be used before their real declaration

const

  • Variables declared with const keywords are also block scoped. Once a const variable is declared, it can never be changed.

Tell us the use cases of null and undefined

null

  • null keyword shows that something is missing a value. It can sometimes also be mean as empty

undefined

  • undefined means that a variable is declared but uninitialized or the property is non-existent.

What do you mean by REST API?

  • A REST API is a type of API which follows some design principles of represented state transfer architectural style. Those web APIs which follows the REST framework are known as REST API.
